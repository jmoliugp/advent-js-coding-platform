## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v7 or higher)

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/25-days-of-coding.git
cd 25-days-of-coding
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
